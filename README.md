```
How to run FastAPI (BackEnd)

# change directory first
$: cd BackEnd/  


# Create venv (Optional)
$: python -m venv .venv
# Activate it (PowerShell)
$: .venv\Scripts\Activate.ps1
# Or if using Command Prompt (cmd)
$: .venv\Scripts\activate.bat
# Or if based and using Linux (Chad)
$: source .venv/bin/activate


# Install requirements
$: pip install -r requirements.txt

# start server
$: uvicorn code:Main:app --port 8080
```

```
How to run Svelte (Frontend)

# change directory first
$: cd Frontend/  

# Install packages inside directory
$: npm install

# Start server
$: npm run dev
```
